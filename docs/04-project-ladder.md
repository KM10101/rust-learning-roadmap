# 项目阶梯

## 项目 1：字符串与文件处理 CLI

做一个小型 CLI，具备这些能力：

- 读取文件
- 统计行数 / 单词数 / 字符数
- 可选地按关键字过滤行

它的价值在于：

- 练文件 IO
- 练参数解析
- 练字符串处理
- 在真实代码里感受所有权基础

## 项目 2：JSON API 获取 CLI

做一个 CLI，具备这些能力：

- 调用公开 HTTP API
- 解析 JSON
- 格式化输出结果
- 做好错误处理

它的价值在于：

- 学会使用 reqwest
- 学会使用 serde
- 理解基于 Result 的流程
- 建立实用的错误处理习惯

## 项目 3：Markdown 知识整理工具

做一个工具，具备这些能力：

- 扫描 markdown 文件
- 提取标题 / 链接 / 关键词
- 输出一份汇总报告

它的价值在于：

- 练 struct 与 enum
- 练文件遍历
- 练模块化设计
- 练解析逻辑测试

## 项目 4：小型任务 API 服务

做一个小型后端服务，具备这些能力：

- 提供 CRUD 接口
- 做输入校验
- 初期先把数据存内存
- 后续再扩展持久化存储

它的价值在于：

- 建立 async 后端直觉
- 理解路由
- 理解共享状态
- 练 API 错误设计

## 项目 5：面向 Agent 的辅助工具

做一个和当前兴趣方向相关的工具，例如：

- prompt 历史整理器
- 仓库 diff 总结器
- markdown 知识索引器
- 日志 / 事件解析器

它的价值在于：

- 更容易保持学习动力
- 能把 Rust 和真实工作连接起来
- 有助于形成长期熟练度
