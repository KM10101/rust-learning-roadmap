# Rust Learning Roadmap

This repository is now rendered with **VitePress** for easier local browsing.

## Local development

```bash
npm install
npm run docs:dev
```

Then open the local VitePress URL in your browser.

## Build

```bash
npm run docs:build
```

## Main content

The documentation site lives under `docs/`.

- `docs/index.md` — site homepage
- `docs/.vitepress/config.mts` — VitePress config
- `docs/01-why-rust.md` to `docs/06-rust-for-backend-engineers.md` — core content
